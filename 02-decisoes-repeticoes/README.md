# Decisões e Repetições em Python

## Sobre o projeto

Este projeto foi desenvolvido durante o bootcamp Python para Análise e Automação de Dados da Accenture em parceria com a DIO.

O objetivo desta etapa foi praticar estruturas condicionais e estruturas de repetição utilizando Python.

Durante o desenvolvimento foram aplicados conceitos fundamentais da lógica de programação, como:
- tomada de decisão;
- validações;
- controle de fluxo;
- repetições;
- interação com o usuário.

O projeto simula um pequeno sistema de validação de acesso utilizando estruturas básicas da linguagem Python.

---

## Conceitos praticados

- if
- elif
- else
- while
- for
- input
- validação de dados

---

## Objetivo de aprendizagem

Desenvolver raciocínio lógico e compreender como sistemas utilizam estruturas condicionais e de repetição para controlar comportamentos e automatizar processos.
